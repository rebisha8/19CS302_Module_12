## Program:
```
/*
  Struct Node
{
int data;
struct Node *next;
}*head;
void display()
{
struct Node *temp= head; 
while(temp!=NULL)
{
printf("%d\n",temp->data); 
temp=temp->next;
}
}

*/
```
